修改提交注释信息

如何修改现有的，未被提交的提交？

git commit --amend

OR

git commit --amend -m "New commit message"

如果您已经推你的提交到你的远程分支,那么就需要强行提交

git push <remote> <branch> --force

OR

git push <remote> <branch> -f

警告：强制推送会用本地状态覆盖远程分支。如果远程分支上有您在本地分支机构没有的提交，那么您将丢失这些提交。

更多详情: stackoverflow
